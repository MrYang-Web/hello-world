      var a = 123;
      var b = a;
      console.log(a);
      console.log(b);
      此时打印的值都为123.

      var a = 123;
      var b = a;
      a++;
      console.log(a);
      console.log(b);
      此时打印的 a 为124
                b 为123

      说明,一个值的变化并不影响另一个值,他们都是独立的.



      var obj =  new Object();
      obj.name = "孙悟空";
      var obj2 = obj;
      obj.name = "猪八戒";
      console.log(obj);
      console.log(obj2);
      此时都将打印孙悟空,说名一个值的变化,另一个值随之变化.

这就是基本数据类型和引用数据类型的区别.

      基本数据类型:
      JS中的变量都是保存在栈内存中.
      基础数据类型的值,直接在栈内存中存储.
      值与值之间是独立存在的,修改一项,并不对其他项造成影响.

      ----------------------------------------------

      引用数据类型:
      Js中的引用数据类型都是保存在堆内存中.
      没创建一个对象,就会在堆内存中创建一个空间,
      而变量中保存的是对象的地址.如果两个对象保存的是对象的同一个地址.
      当一个通过修改对象的属性时,另一个也就随之改变了.

      var obj =  new Object();
      obj.name = "孙悟空";
      var obj2 = obj;
      obj2=null;
      console.log(obj);
      console.log(obj2);
      当设置obj2的值为空的时候,并不会造成obj打印结果为空.
      这里只是让obj2断开了和堆内存的链接.

      ---------------------------------------------

      var a = 10;
      var b = 10;
      console.log(a == b);   将打印结果true;


      var obj3 = new Object();
      var obj4 = new Object();
      console.log(obj3 == obj4); 打印的结果将为false;


      当比较两个基本数据类型的值时,就是比较值.
      当比较两个引用数据类型时,它是比较数据的内存地址.
      就算两个对象值一摸一样,地址不一样时,也会返回false.
