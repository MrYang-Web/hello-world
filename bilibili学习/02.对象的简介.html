Javascript中的基本数据类型

  • string 字符串
  • number 数值
  • boolean 布尔值
  • null 空值
  • undefind 未定义

以上5中是javascript中的基本数据类型,以后我们看到的值只要不是以上五种,那么就是object!

基本数据类型之间没有直接的联系

object 对象

对象的分类

  • 内建对象

    由ES标准中定义的对象,在任何的ES中都可以实现使用.

    比如:Math String Number Boolean FUnction Object....等等

  • 宿主对象

    由Javacsript运行环境提供的对象,目前来讲主要指浏览器提供的对象.

    比如:BOM DOM

    console document 这两个都是由浏览器创建提供

  • 自定义对象

    由开发人员自己创建的对象.
