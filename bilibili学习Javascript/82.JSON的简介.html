82.JSON的简介

JSON

JS中的对象,只有JS自己认识,其他语言无法识别

JSON就是一个特殊格式的字符串,这个字符串可以被任意的语言认识.

并且可以转换为任意中的对象,JSON在开发中主要负责数据的交互和传递

Javascript Object Notation

JS对象表示法

JSON和JS的对象格式一样,只不过JSON字符串对象中的属性名,必须加双引号

JSON的分类

1.对象

2.数组

JSON中允许的值

1.字符串 2.数值 3.布尔值 4.null 5.数组 6.对象(普通数组,不包括函数)

			var obj = '{"name":"孙悟空","age":18,"gander":"男"}';
			
			var arr ='{1,2,3,"hello",true}';

将JSON对象转换为JS对象

在JS中为我们提供了一个JSON工具类

这个对象可以帮我们将一个JSON转换为JS对象,也可以将一个JS对象转换为JSON

JSON ---> JS

JSON.parse(); 这个方法可以将JSON字符串转换为JS对象

它需要一个JSON字符串作为参数,会将该字符串转换为JS对象返回.

JS ---> JSON

JSON.stringify();这个方法可以将JS对象转换为JSON字符串

它需要一个js对象作为参数,会返回一个JSON字符串
