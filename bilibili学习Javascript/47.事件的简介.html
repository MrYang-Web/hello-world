47.事件的简介

事件,就是文档或浏览器窗口中发生的一些特定的交互瞬间

Javascript与HTML之间的交互是通过事件实现的

对于web应用来说,有下面这些代表性的事件:点击某个元素,将鼠标移至某个元素上方,按下键盘某个键,等等

事件并不关键,关键的是事件发生以后,要有行为反馈.

      我们可以在JavaScript事件中设置一些代码,这样当事件被触发时,这些代码将会执行.

这种写法称为结构和行为耦合,不方便维护,不推荐使用

我们可以为按钮对应事件绑定处理函数的形式来响应事件

      获取按钮对象
      var btn = document.getElementById("btn");

      再绑定一个处理函数.
      btn.onclick = function(){
        alert("讨厌,你还点");
      }
      像这种为单击事件绑定的函数,我们称为单击回调函数.
