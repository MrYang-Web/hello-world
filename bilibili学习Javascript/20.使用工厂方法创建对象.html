20.使用工厂方法创建对象

创建一个对象

      var obj = {
        name:"孙悟空",
        age:18,
        gander:"男",
        sayName = function(){
          alert(this.name);
        }
      };

当我们需要创建N个类似的obj,就可以用工厂方法来创建

通过该方法可以大批量创建对象

这里因为是通过new关键词创建的对象,所以在添加属性的时候,要在前面添加obj.

这个时候就可以批量的创建所需要的对象了

      var obj2 = createPerson("孙悟空",18,"男");
      var obj3 = createPerson("白骨精",18,"女");

注意点:使用工厂方法创建的对象,使用的构造函数都是Object

所以创建的类型都是Object这个类型
