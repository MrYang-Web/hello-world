41.正则表达式的简介

一般用于验证用户输入的信息是否有误

比如电子邮件的输入格式 abc@abc.com

1.前面什么都可以

2.跟着一个@

3.后边什么都可以

4. .com获取其他的乱七八糟

正则表达式用于定义一些字符串的规则

计算机可以根据正则表达式,来检查一个字符串是否符合规则

或者将字符串中符合规则的内容提取出来

创建正则表达式对象

语法:

      var 变量 = new RegExp("正则表达式","匹配模式");

正则表达式方法test()

使用这个方法可以用来检查一个字符串是否符合正则表达式的规则

如果符合则返回true,否则返回false.

      var reg = RegExp("a");  //这个正则表达式可以检查字符串中是否含有"a"
      var str = "abcdertg";
      var result = str.test(str);
      console.log(result);

在构造函数中可以传递一个匹配模式作为第二个参数

可以是 i g

i 表示的是忽略大小写

g 表示的是全局匹配模式

      var reg = RegExp("a","i");

使用字面量创建正则表达式

语法: var 变量 = /正则表达式/匹配模式

      var reg = /a/i;

创建一个正则表达式来检查字符串中是否含有 a或者b或者c

      var reg = /a|b|c/;

      var reg = /[abc]/;  //这种写法与上述写法一样;

使用一个 | 竖线来表示逻辑或

创建一个正则表达式检查一个字符串中是否含有字母

      var reg = /[A-z]/;   //这个正则表示任意的小写字母;

[]里的内容也是或的关系

[a|b] == a|b

[a-z] 任意小写字母

[A-Z] 任意大写字母

[A-z] 任意字母

[0-9] 任意数字

检查一个字符串中是否含有abc adc aec

      var reg = /a[bde]c/;

[^ab] 表示取反,除了a或b都行
