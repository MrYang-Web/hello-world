28.数组的字面量

数组也是对象

但是如果数组不储存数字,就没有意义了.

      var arr = new Array();
      arr[0] = 1234;
      arr.hello = "helo world";
      console.log(arr.hello);

这里是可以打印hello world的.

数组也可以用字面量的形式来创建

语法: 数组 = []

      var arr2 = [1,2,3,4,5,6,10];

      console.log(arr2);

会打印出数组

第二种写法:使用构造函数创建数组时,也可以同时添加元素

将要添加的元素作为构造函数的参数传递

      var arr3 =new Array(10,20,30);

这里有一个主意点:

      创建一个数组中只有一个元素10
      arr4 = new Array(10);
      console.log(arr4);

会打印一串逗号...所以这个10其实是表示这个数组的长度是10

数组中的元素可以是任意数据类型


      arr5 = ["hello",1,true,null,undefind];
      console.log(arr5);

也可以是对象

      var obj = {name:"孙悟空"};
      var[arr5.length] = obj;
      console.log(arr5[5].name);
      打印结果为孙悟空;

      arr6 = [{name:白骨精},{name:沙和尚},{name:蜘蛛精}];

      console.log(arr6[0],arr6[2]);

数组里的元素也可以在是一个函数


      arr7 = [function(){},function(){},function(){}];

也可以调用: arr7[0]();

数组里的元素也可以是数组(二维数组)


      arr8 =[[1,2,3],[4,5,6],[7,8,9]];
