42.与正则表达式相关的字符串方法

split()方法,可以将一个字符串拆分我数组

      var str = "1a2b3c4d5e6f7A8B9C0)";
      var result = str.split("c");
      console.log(result);

根据任意字母拆分

方法中可以传递一个正则表达式作为参数

方法就会根据正则表达式拆分字符串

这个方法即使不进行全局匹配也会进行全局匹配

      var result2 = str.split(/[A-z]/);

search()可以检索字符串中是否含有正则表达式的指定内容

      var str2 = "123 456 abc 789 def";
      var result3 = str2.search(/abc/);

如果搜索到指定内容,则会出现第一次出现的索引

如果没有则返回-1

search 只会查找第一个,即使设置全局匹配也没用

搜索abc或aec或afc

      var str3 = "abc 123 aec 456 afc 789";
      var result4 = str3.search(/a[bef]c);

match()可以根据正则表达式,从一个字符串中将符合条件的内容提取出来

默认情况下match()只会找到第一个符合要求的内容,找到后就停止检索

我们可以设置正则表达式为全局匹配模式

      var result5 = str.match(/[A-z]/);  //提取一个
      var result6 = str.match(/[A-z]/g);  //全局模式,提取所有

match()会将返回的值封装到一个数组对象内,即使只查询到一个值.

replace()可以将字符串的内容替换为指定的内容

参数一: 被替换的内容(可以使用正则表达式)

参数二: 替换的内容

      var result7 = str.replace("a","@_@");

默认只会替换第一个,如果要替换全部英文字母,就需要使用正则表达式

      var result8 = str.replace(/[A-z]/g,"@_@");
