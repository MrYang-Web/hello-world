forEach方法的使用

一般我们都是用for方法遍历数组

JS还为我们提供了forEach()方法,用来遍历数组

      var arr = ["孙悟空","猪八戒","沙和尚","唐僧","白骨精"];

forEach需要一个函数作为参数

像这种函数,由我们创建,但是不由我们调用,我们称为回调函数

      arr.forEach(function(){
        console.log("hello");
      });

数组中有几个元素,函数就会执行几次.每次执行时,浏览器会将遍历到的元素以实参的形式传递进来.

浏览器会在回调函数中传递三个函数.

      arr.forEach(function(a,b,c){
        console.log("a =" + a);
        console.log("b =" + b);
        console.log("c =" + c);
      });

第一个元素是遍历的元素value

第二个元素是遍历到元素的索引index

第三个元素是遍历的数组arr
