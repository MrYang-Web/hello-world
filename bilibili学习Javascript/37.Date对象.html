Date对象

在Js中使用Date对象表示一个时间

      创建一个时间对象
      如果直接使用构造函数创建一个Date对象,则会封装为当前代码执行的时间.
      var d = new Date();
      console.log(Date);

创建一个指定的时间对象

需要在构造函数中传递一个表示时间的字符串

var d2 = new Date("12/03/2018 11:10:30"); console.log(d2);

日期的格式: 月份/日期/年份 时:分:秒

Date的方法

getDate()方法获取当前日期对象时几号

getDay()方法获取当前日期是周几,会返回一个0-6的值,0表示周日

getMonth()方法获取当前日期是周几,会返回一个0-11的值,0表示1月

getTime()方法获取当前日期对象的时间戳.指的是从格林威治时间1970年1月1日,0时0分0秒

方便我们统一计时,计算机底层都是使用时间戳保存时间.

可以利用时间戳来测试代码的性能

      var start = Date.now();

      for(i=0;i < 1000; i++){
        console.log(i);
      }
      var end = Date.now();

      console.log("执行了:"+(start-end)+"毫秒");
