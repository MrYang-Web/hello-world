33.数组去重练习作业

      var arr = [1,2,3,2,1,3,4,2,5];

      删除重复的元素,要使用的splice和for循环的嵌套

1.获取数组的每一个元素

      for(i=0; i< arr.length; i++){

      }

2.让每一个元素跟数组中所有元素进行比对.重复就删除.


        for(var i=0; i< arr.length; i++){

          for(var j=i+1; i< arr.length; j++){
            if(arr[j] == arr[i]) {
              arr.splice(j,1);
              j--;
            }
          }
        }
        console.log(arr);

当删除了j所在的元素后,后面的元素将自动补位.

此时将不会再比较这个元素,需要再比较一次j所在的位置.
