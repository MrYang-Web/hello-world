25.jQuery的冒泡和默认行为

1.什么是事件冒泡?

事件从下级往上级传递就叫做事件的冒泡

2.如何阻止事件冒泡?

1.只需要在子级事件中return false就可以阻止事件冒泡

	$(".son").click(function(){
		alert("son");
		return false;
	})

2.第二种方法.只要事件触发了回调函数,都会给回调函数中传入一个参数event

这个event对象保存了事件对象的状态.只有当函数执行时,才有会.

	$(".son").click(function(event){
		alert("son");
		event.stopPropagation();
	})

3.什么是默认行为?

在html中很多标签默认存在一些行为,很多时候,我们不想要这种默认行为比如a标签的跳转

4.如何阻止默认行为

只需要选中元素,然后再回调函数中设置return false就可以阻止默认事件

案例中的a标签,阻止了默认跳转至百度页面

	$("a").click(function(){
		alert("弹出注册框");
		return false;
	})

同样的,阻止默认行为也有两种方法

	$("a").click(function(event){
		alert("弹出注册框");
		event.prevntDefault();
	})
