什么是静态方法和实例方法

直接添加到类上面的就是静态方法,直接添加到原型上的就是实例方法!

1.静态方法通过类名调用

2.实例方法通过类的对象调用

1.定义一个类 AClass

	function AClass() {
		
	}

2.给这个类添加一个静态方法

直接添加给类的就是静态方法

	AClass.staticMethod = function() {
		alert("staticMethod");
	}

静态方法通过类名调用

	AClass.staticMethod();

3.给这个类添加一个实例方法

	AClass.prototype.instancMethod = function () {
		alert("instancMethod");
	}

实例方法通过类的实例调用

创建一个实例(创建一个对象)

	var a = new AClass();
	a.instancMethod();
