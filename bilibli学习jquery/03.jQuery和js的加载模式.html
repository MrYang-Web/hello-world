jQuery的加载模式

		1.Javascript原生入口函数:
			window.onload = function (){
				通过原生JS的入口函数可以拿到DOM元素.
				var img = document.getElementsByTagName("img")[0];
				console.log(img);
				通过原生JS的入口函数可以拿到DOM元素宽.
				var width = window.getComputedStyle(img).width;
				console.log(width);
			}

		2.jQuery入口函数:
			$(document).ready(function(){
					通过jQuery的入口函数可以拿到DOM元素.
					var $img = $("img");
					console.log($img);
					通过jQuery的入口函数可以拿到DOM元素的宽度.
					var $width = $img.width();
					console.log("ready", $width);
				});

		jQuery 的写法相对比较简单
					 语义化也比较强
		原生JS和jQuery的加载方式不一样
		原生JS会等到DOM元素加载完毕,并且图片也加载完毕才会执行.
		jQuery会等到DOM元素加载完毕,但不会等图片加载完毕才执行.


		3.Javascript如果编写了多个入口函数,后面编写的会覆盖前面编写的.
		window.onload =function (){
			alert("hello one");
		}
		window.onload =function (){
			alert("hello two");
		}
		此处只会执行hello two

		4.jQuery如果编写了多个入口函数,后面编写的入口函数并不会覆盖前面编写的.
		$(document).ready(function() {
			alert("hello one")
		});
		$(document).ready(function() {
			alert("hello two")
		});
