静态方法-each方法的掌握

原生javascript的each方法

      var arr = [1, 3, 5, 7, 9,];
      var obj = {0:1, 1:3, 2:5, 3:7, 4:9, length:5};
      第一个参数value ,遍历到元素的值
      第二个参数index ,遍历到的索引
      注意点:原生JS只能遍历到数组,不能遍历到伪数组.
        arr.forEach(function (value, index){
        console.log(index,value);
      })

      第一个参数index,遍历到元素的索引.
      第二个参数,遍历到元素的值.
      jQuery可以遍历到伪数组.
      $.each(arr, function (index, value) {
        console.log(index,value);
      })

      $.each(obj, function (index, value) {
        console.log(index,value);
      })
